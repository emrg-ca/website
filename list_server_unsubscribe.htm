How To Unsubscribe

If you want to remove yourself from the list server, prepare an e-mail as shown below;

  1. Select the plain-text option before sending the message
  2. Enter the following address in the TO field: majordomo@igs.net
  3. Leave the subject field blank
  4. In the message area, type: 
        unsubscribe arec your-email-address

    For Example: 
        unsubscribe arec ve3oce@rac.ca
