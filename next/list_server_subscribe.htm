EMRG List Server

How To Subscribe

If you want to join the list server, prepare an e-mail as shown below;
  1. Select the plain-text option before sending the message.
  2. Enter the following address in the TO field: majordomo@igs.net
  3. Leave the subject field blank
  4. In the message area, type:

    subscribe arec your-email-address

    For Example: 

    subscribe arec ve3oce@rac.ca
